# OpenRelax PC Care 🚀

**OpenRelax** is a lightweight, safe, and beautiful open-source Windows system optimization utility built using PowerShell and Windows Forms (WinForms). It is designed to safely reclaim system memory (RAM) and clean temporary application/browser caches without interrupting your active workflow, open tabs, or document states.

---

## Özellikler / Features

### 🇹🇷 Türkçe
- **Tek Tıkla Sistem Bakımı:** Tek butona basarak tüm optimizasyon ve temizlik adımlarını saniyeler içinde tamamlayın.
- **Güvenli RAM Optimizasyonu:** Native Windows API'lerini kullanarak uygulamaların kullanmadığı fiziksel bellek bloklarını işletim sistemine geri kazandırır. Aktif süreçlerinizi kapatmaz veya çökertmez.
- **Kapsamlı Önbellek Temizliği:** Kullanıcı/Sistem Temp klasörlerini, Google Chrome, Microsoft Edge, Brave, Opera Stable, Mozilla Firefox ve Discord önbelleklerini ve DirectX Shader önbelleğini temizler.
- **Otomatik RAM Boşaltma:** Arka planda çalışarak RAM doluluk oranı belirlediğiniz eşiğe (örn. %75, %80, %85, %90) ulaştığında otomatik ve sessizce optimizasyon yapar.
- **Gerçek Zamanlı Monitör:** CPU ve RAM kullanımı, aktif süreç sayısı ve bilgisayar çalışma süresini (Uptime) anlık görüntüler.

### 🇺🇸 English
- **One-Click Maintenance:** Run all memory boosts and cache cleanups in seconds with a single action.
- **Safe RAM Optimization:** Utilizes native Windows APIs to reclaim physical RAM allocated to idle processes without terminating active windows or browser tabs.
- **Comprehensive Cache Cleaning:** Targets user/system Temp directories, Google Chrome, Microsoft Edge, Brave, Opera Stable, Mozilla Firefox, Discord caches, and DirectX Shader cache.
- **Auto-Boost RAM:** Automatically and silently optimizes memory when utilization exceeds your custom threshold (e.g. 70%, 75%, 80%, 85%, 90%).
- **Real-Time Diagnostics:** Displays live CPU/RAM usage, process counts, and system uptime.
